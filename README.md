📘 Project README

🧠 Python Mini Projects Collection

This repository contains three beginner-to-intermediate Python projects:

 Factorial Calculator (Level 1)

 Fibonacci Sequence Generator (Level 2)

 Tic-Tac-Toe Game – CLI Based (Level 3)



🚀 1. Fibonacci Sequence Generator

📌 Description

This program generates the Fibonacci sequence up to a user-defined number of terms.

⚙️ How It Works

Starts with two numbers: 0 and 1

Each next number is the sum of the previous two

Uses a loop to generate the sequence



🧾 Example Output

Enter number of terms: 5
Fibonacci sequence: [0, 1, 1, 2, 3]



🎮 2. Tic-Tac-Toe Game (CLI)

📌 Description

A two-player command-line Tic-Tac-Toe game where players take turns entering positions.

⚙️ Features

Interactive board display

Input validation (prevents overwriting moves)

Winner detection (rows, columns, diagonals)

Draw detection


🎯 How to Play

Players take turns (X and O)

Enter a number from 1–9 corresponding to board positions

First player to align 3 symbols wins


🧾 Example Board

1 | 2 | 3
4 | 5 | 6
7 | 8 | 9




🧮 3. Factorial Calculator

📌 Description

Calculates the factorial of a given number.

⚙️ Formula

n! = n × (n-1) × (n-2) × ... × 1

🛠️ Technologies 

Command Line Interface (CLI)
